A Simple plugin to generate Notice Text using Widget.

== Description ==

Sticky Note by Dolar Patel is an easy to use widget plugin with Shortcode to generate Notice Text that you can use in any sidebar.

No need of authentication such as password and API keys Generation for Sticky Note Plugin. 

We are not making any call backs to our server and not storing any data of user.

= Key Features =

* Control for specify number of paragraph length.

*Parameter 

	* Title Color - Define Title Text Color.

	* Notice Text Color- Define Notice Text Color.
	
	* Background Picture Type


== Installation ==

1. Unzip 'widget_plugin.zip' to the '/wp-content/plugins/' directory or add it by 'Add new' in the Plugins menu.
2. Activate the plugin through the 'Plugins' menu.
3. Go to Appearance->Widgets.
